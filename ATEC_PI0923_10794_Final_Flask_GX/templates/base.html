Objetivo do Projeto

Este trabalho tem como objetivo desenvolver um sistema de gestão de viagens temporais, permitindo criar reservas, gerir perfis de viajantes, consultar viagens marcadas e efetuar pedidos personalizados. O projeto demonstra o uso de Flask, templates Jinja, rotas dinâmicas e manipulação de dados em memória.
